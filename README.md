# Prosperity Prognosticator: Machine Learning for Startup Success Prediction

Prosperity Prognosticator is a machine learning project designed to predict the success probability of startups using historical business data. The system analyzes factors such as funding amount, number of funding rounds, industry category, and startup age. Data preprocessing techniques are applied to clean and prepare the dataset for modeling. Multiple machine learning algorithms like Logistic Regression, Decision Tree, and Random Forest are implemented. The models are trained to identify patterns that influence startup growth and sustainability. Performance is evaluated using metrics such as accuracy, precision, recall, and F1-score. The project demonstrates how data-driven approaches can support investment and business decisions. It provides insights into key features that impact startup success. The system can help entrepreneurs and investors make informed choices. This project showcases the practical application of machine learning in business analytics.
